Provides a series of classes that implement P4Java-wide server, client, and other class method options definitions and processing.

Package com.perforce.p4java.options and its changelist, client, and server sub-packages were introduced with the 2010.1 release to make it easier for users and developers to ensure forwards and backwards binary compatibility. Please see the Perforce knowledge base article linked to here (TBD) for the full semantics and usage details.
